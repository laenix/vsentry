# VSentry Frontend

React-based management console for VSentry SIEM + SOAR platform.

## Setup

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Features

- Dashboard with system overview
- Log query with Monaco editor
- Detection rules management
- Incident tracking and response
- SOAR automation with visual workflow editor
- Ingest endpoint management with token generation
- Collector builder for Windows/Linux/macOS
- Third-party integrations (Connectors)
- Custom table definitions
- User management and settings
