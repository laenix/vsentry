### Configuration

To customize the deployment:

```bash
# Update external URL
helm upgrade {{ .Release.Name }} vsentry/vsentry \
  --set config.externalUrl=https://vsentry.yourdomain.com

# Use external VictoriaLogs
helm upgrade {{ .Release.Name }} vsentry/vsentry \
  --set victorialogs.enabled=false \
  --set externalVictorialogs.url=http://victorialogs:9428
```
